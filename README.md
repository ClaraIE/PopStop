# PopStop

PopStop listens to microwave popcorn, detects when popping slows, and warns the user before the bag turns into evidence.
It is for adults who want perfect popcorn but do not trust themselves, their microwave, or the legacy popcorn industry.

PopStop is built around a simple cooking moment: set up the microwave once, photograph the bag if useful, tap Start, and let the app listen locally. Audio stays on the device. No recordings, no accounts, no cloud analysis. When pop cadence collapses, PopStop triggers a loud alarm, haptics, and a blunt voice message telling the user to kill the microwave.
